# sales_taxes

### How to run
1. Use ruby version 2.7.1

2. Install bundler if it's not already installed
```shell
gem install bundler
```

3. Install gems
```shell
bundle install
```

4. Invoke Rake file
```shell
rake
```

5. Input each line with hitting enter key on keyboard, and end input by hitting enter without any input

6. See the result!
